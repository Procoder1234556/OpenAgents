# Submission Checklist

## Required links

- Repository URL
- Deployed app URL
- Demo video URL

## Devpost / submission form fields

- Title: `PulsePilot`
- Summary: use the abstract from `SUBMISSION.md`
- Technologies: Next.js, React, TypeScript, Dynatrace MCP, Gemini API, Zod, custom CSS

## Before publishing

- Confirm the deployed app opens successfully.
- Confirm the video is public and under 3 minutes.
- Confirm the README links are correct.
- Confirm the submission abstract matches the actual build.
- Confirm the repository has a visible open-source license.
