# Submission Package

## Title

PulsePilot

## Summary abstract

PulsePilot is a Dynatrace-powered incident command center that converts active production signals into a safe, structured response plan. The app ingests incident context, correlates observability evidence, generates a root-cause hypothesis with optional Gemini drafting, and surfaces approval-gated remediation steps in a polished web UI.

## Technologies used

- Next.js
- React
- TypeScript
- Dynatrace MCP
- Gemini API
- Zod
- Custom CSS
